# Proyek: Todo List Interaktif (JavaScript Murni)

Proyek *Todo List* ini adalah studi kasus untuk menunjukkan penguasaan dasar-dasar **DOM Manipulation**, **Event Handling**, dan **Manajemen Data Sisi Klien** menggunakan JavaScript.

Ini adalah jenis proyek yang relevan untuk *micro-job* yang memerlukan *utility* berbasis browser tanpa *server* yang kompleks.

* **Counter Statistik:** Menampilkan hitungan *Pending*, *Completed*, dan *Total* item.
